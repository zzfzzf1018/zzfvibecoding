## 2. 数据源适配层（详细）

### 2.1 interfaces.py（端口）
```python
from typing import Protocol, runtime_checkable
@runtime_checkable
class ValuationSource(Protocol):
    name: str
    def get_latest(self, index_code: str) -> "IndexValuation | None": ...
    def get_history(self, index_code: str, start: date, end: date) -> "list[IndexValuationPoint]": ...

@runtime_checkable
class ConstituentSource(Protocol):
    name: str
    def get_constituents(self, index_code: str) -> "list[ConstituentStock]": ...

@runtime_checkable
class EtfBasicSource(Protocol):
    name: str
    def list_etfs(self) -> "list[EtfBasic]": ...
```
> 所有对外类型用 `models/schemas.py` 的 Pydantic/Dataclass 定义，避免 ORM 对象泄漏到数据源层。

### 2.2 akshare_src.py（主实现，示意）
```python
class AkShareValuationSource:
    name = "akshare"
    def get_latest(self, index_code: str) -> IndexValuation | None:
        df = ak.stock_index_value_name_em()          # 指数估值总表
        row = df[df["指数代码"] == index_code]
        if row.empty: return None
        return IndexValuation(index_code=index_code, pe_ttm=row.pe, pb=row.pb,
                              dividend_yield=row.dy, valuation_date=row.date, source=self.name)
    # get_history: 优先本地 history；外部历史由 ValuationRepo 补录
```
- 失败处理：捕获异常 → 返回 None（由 services 切兜底）。
- 限流：`core/clients/http_client` 统一限速（见 §6）。

### 2.3 csindex_src.py / em_src.py
- 同结构实现 `ValuationSource` / `ConstituentSource`，作为兜底。
- 中证：解析官网 JSON；东财：复用 AkShare 已封装或直连。

## 3. 业务层（services）详细

### 3.1 SearchService（FR-01）
```python
class SearchService:
    def __init__(self, etf_repo: EtfRepo): ...
    def search(self, keyword: str, page=1, size=50) -> list[ETFSearchResult]:
        if not keyword: return []
        if keyword.isdigit():                       # 编号匹配
            return etf_repo.search_by_code(keyword, page, size)
        return etf_repo.search_by_name(keyword, page, size)   # 模糊(ILIKE/子串)
```
- 排序：宽基优先 + fund_scale 降序。

### 3.2 ValuationService（FR-03）
```python
class ValuationService:
    def __init__(self, etf_repo, valuation_repo, sources: list[ValuationSource]):
        self.sources = sources          # 主→兜底 顺序注入
    def get_valuation(self, code: str) -> ValuationView:
        etf = etf_repo.get_basic(code) or raise NotFound
        if etf.type in (债券, 货币, 商品):               # 不适用类
            return ValuationView(applicable=False, reason="该类 ETF 无指数估值")
        cached = valuation_repo.get_latest(etf.track_index_code)
        if cached: return to_view(cached)
        for src in self.sources:                       # 降级
            v = safe(src.get_latest, etf.track_index_code)
            if v: valuation_repo.upsert_latest(v); return to_view(v)
        raise SourceUnavailable
```

### 3.3 PercentileService（FR-04）
```python
def get_percentile(self, code, window: Window) -> PercentileView:
    etf = etf_repo.get_basic(code)
    hist = valuation_repo.get_history(etf.track_index_code, window_start(window))
    if len(hist) < 250: window = degrade(window)        # 样本不足降级
    cur = valuation_repo.get_latest(etf.track_index_code)
    pe_pct = percentile_rank([h.pe for h in hist], cur.pe_ttm)
    pb_pct = percentile_rank([h.pb for h in hist], cur.pb)
    return PercentileView(pe_percentile=pe_pct, pb_percentile=pb_pct,
                          pe_max=max(pe), pe_min=min(pe), pe_median=median(pe), ...)
```
**分位算法**：
```
percentile_rank(series, x) = count(v <= x for v in series) / len(series) * 100
```
- 最小样本阈值 250；窗口 `1y/3y/5y/all` → `window_start` 计算起始日期。
- 估值缺失/异常（pe<=0）跳过该样本。

### 3.4 ConstituentService（FR-05）
```python
def get_constituents(self, code, top_n=None, industry=None) -> list[ConstituentStock]:
    etf = etf_repo.get_basic(code)
    rows = constituent_repo.get_constituents(etf.track_index_code)
    if industry: rows = [r for r in rows if r.sw_l1 == industry]
    rows.sort(key=lambda r: r.weight, reverse=True)
    if top_n: rows = rows[:top_n]
    return rows
def industry_distribution(self, code) -> list[IndustryWeight]:
    rows = get_constituents(code)
    agg = groupby_sw_l1_sum_weight(rows)
    return agg
```

## 5. 错误处理与错误码
```python
class AppError(Exception):
    code: int; message: str
class NotFound(AppError): code=404
class SourceUnavailable(AppError): code=503
class InvalidParam(AppError): code=400
```
| code | 场景 | 处理 |
| --- | --- | --- |
| 400 | 参数非法 | 返回 message |
| 404 | ETF 代码不存在 | 友好提示 |
| 503 | 所有数据源不可用 | 不抛 500，降级提示 |
| 429 | 触发限流 | 返回 Retry-After |

## 6. 横切：core 详细
- `config.py`：pydantic-settings 读环境变量（`DATA_SOURCE_PRIMARY`, `CACHE_TTL_*`, `HTTP_TIMEOUT`, `RATE_LIMIT`）。
- `logging.py`：结构化日志，含 `source`/`index_code` 上下文。
- `clients/http_client.py`：带超时(10s)、重试(≤3,指数退避)、限速(间隔≥0.3s) 的受控客户端；仅 datasource 使用。
- `di.py`：装配具体源 → 列表注入 services；一处切换主/兜底顺序。**测试在此替换为 Mock。**

## 7. Scheduler 详细
```python
def refresh_etf_basic():     # 每日，增量 upsert
    for e in primary_basic_source.list_etfs(): etf_repo.upsert_basic(e)
def refresh_valuation():     # 交易日 T+1
    for idx in distinct_track_index(): 
        v = pick_source().get_latest(idx)
        if v: valuation_repo.upsert_latest(v); valuation_repo.append_history(today, v)
def refresh_constituents():  # 7d 或公告触发
    for idx: constituent_repo.replace_constituents(pick_source().get_constituents(idx))
```
- 失败：单指数失败不影响整体；记录日志 + 告警计数。

## 8. 序列图（估值查询）
```
UI → API: GET /valuation/{code}
API → ValuationService.get_valuation(code)
ValuationService → EtfRepo.get_basic → 返回 etf(index_code)
ValuationService → ValuationRepo.get_latest → 命中? 返回
  否 → 遍历 sources[i].get_latest → 成功则 upsert + 返回
  全失败 → raise SourceUnavailable
ValuationService → API: ValuationView
API → UI: JSON
```

## 10. 松耦合落实检查点
- `services/` 仅 `from app.datasource.interfaces import ...`；严禁 `import akshare`。
- 具体源仅在 `di.py` 被引用；新增源不动 services/api。
- repositories 是唯一 SQL 入口。
- 所有外部 HTTP 经 `core/clients/http_client`。
