# 架构设计文档

## 整体架构

```
┌─────────────────────────────────────────────────────┐
│                    UI 层 (app.py)                     │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐           │
│  │ 搜索组件  │  │ 详情展示  │  │ K线/图表  │           │
│  └──────────┘  └──────────┘  └──────────┘           │
├─────────────────────────────────────────────────────┤
│               Service 层 (services/)                  │
│  ┌────────────────┐  ┌───────────────────┐          │
│  │  ETFService     │  │ ValuationService  │          │
│  │  聚合 ETF 数据  │  │  PE/PB 分位数计算  │          │
│  └────────────────┘  └───────────────────┘          │
├─────────────────────────────────────────────────────┤
│                API/Data 层 (api/)                     │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐          │
│  │ ETF 信息  │  │ 持仓数据  │  │ 估值数据  │          │
│  │ akshare  │  │ akshare  │  │ akshare  │          │
│  └──────────┘  └──────────┘  └──────────┘          │
└─────────────────────────────────────────────────────┘
```

## 数据流

1. 用户在 UI 输入 ETF 代码/名称
2. UI 调用 ETFService.search_etf(keyword)
3. ETFService 调用 API 层获取数据
4. API 层通过 akshare 从东方财富等数据源获取数据
5. Service 层聚合、计算 PE/PB 分位数
6. 结果返回 UI 层渲染展示

## 模块说明

### UI 层
- `app.py` - Streamlit 主入口，包含搜索和 Tab 切换
- `src/ui/components.py` - 可复用的 UI 组件（K线图、持仓表等）

### Service 层
- `src/services/etf_service.py` - ETF 数据聚合服务，组合多个 API 调用
- `src/services/valuation_service.py` - 估值计算服务，计算分位数

### API 层
- `src/api/etf_data.py` - 封装 akshare 调用，获取 ETF 各类数据

### 配置
- `src/config.py` - 全局配置常量

## 数据源

使用 akshare 库，数据来源为东方财富、天天基金等公开金融数据接口。

## 技术栈
- Python 3.10+
- Streamlit - Web UI
- akshare - 金融数据
- plotly - 交互式图表
- pandas - 数据处理
