# 需求规格说明书 — 林中路阅读器 (Holzwege Reader)

> 版本: 1.6.0 | 更新日期: 2026-05-31

## 1. 产品概述

### 1.1 产品目标

为中文用户提供简洁、稳定的 EPUB 阅读体验，支持 TTS 朗读，兼容 HarmonyOS。

### 1.2 目标用户

- 使用 Android / HarmonyOS 手机的中文电子书读者
- 需要 TTS 朗读功能的用户（通勤、休息时听书）

### 1.3 运行环境

- Android 7.0+ (API 24+)
- HarmonyOS 4（Android 兼容模式）
- 屏幕: 适配手机竖屏

## 2. 功能需求

### FR-001: EPUB 文件导入

- **优先级**: P0
- **状态**: ✅ 已实现 (v1.0)
- **描述**: 用户可通过系统文件选择器导入 .epub 文件到书架
- **验收标准**:
  - 支持 EPUB 2.0 和 3.0 格式
  - 导入后文件复制到应用内部存储
  - 自动提取书名和封面作为书架显示

### FR-002: 书架管理

- **优先级**: P0
- **状态**: ✅ 已实现 (v1.1)
- **描述**: 书架页面显示所有已导入图书，支持打开和删除
- **验收标准**:
  - 列表显示书名
  - 点击打开阅读
  - 左滑显示删除按钮，确认后删除
  - 右下角 FAB 按钮导入新书

### FR-003: 章节阅读与分页

- **优先级**: P0
- **状态**: ✅ 已实现 (v1.6.0)
- **描述**: 以分页形式显示章节内容，支持翻页浏览
- **验收标准**:
  - 每页内容完整显示，不截断任何行
  - 左右滑动或点击左右区域翻页
  - 章节结尾自动切换到下一章
  - 页码指示器显示 "第N页/共M页"
  - 底部不显示属于下一页的部分行

### FR-004: 阅读进度保存

- **优先级**: P0
- **状态**: ✅ 已实现 (v1.0)
- **描述**: 自动保存阅读位置，再次打开从上次位置恢复
- **验收标准**:
  - 退出时自动保存（章节+滚动位置）
  - 再次打开恢复到精确位置

### FR-005: 目录导航 (TOC)

- **优先级**: P1
- **状态**: ✅ 已实现 (v1.1)
- **描述**: 显示 EPUB 目录，支持跳转到指定章节
- **验收标准**:
  - 弹窗列表显示所有章节标题
  - 点击跳转到对应章节

### FR-006: 主题与外观

- **优先级**: P1
- **状态**: ✅ 已实现 (v1.2-v1.4)
- **描述**: 多种阅读主题和字体设置
- **验收标准**:
  - 5种主题: 日间(白)、护眼(米黄)、绿色、暗色、纯黑
  - 可调: 字体大小、行高、字间距
  - 可调: 字体颜色（自定义）
  - 可调: 屏幕亮度
  - 设置即时生效（不需重启）

### FR-007: TTS 朗读

- **优先级**: P1
- **状态**: ✅ 已实现 (v1.3-v1.6.0)
- **描述**: 使用系统 TTS 引擎朗读章节内容
- **验收标准**:
  - 通知栏持续显示控制（暂停/继续/上下章/停止）
  - 点击通知回到阅读器
  - 当前朗读句子高亮显示
  - 自动翻页到当前朗读位置
  - 朗读时手动翻页自动暂停
  - 支持从上次位置/当前页/开头开始
  - 语速可调 (0.5x - 3.0x)
  - 进度保存，下次可恢复
  - 自动跨章节连续朗读

### FR-008: 图片查看

- **优先级**: P2
- **状态**: ✅ 已实现 (v1.6.0)
- **描述**: 点击 EPUB 中的图片可放大查看
- **验收标准**:
  - 点击图片弹出全屏覆盖层
  - 支持双指缩放 (1x-5x)
  - 点击覆盖层关闭

### FR-009: 章节内链接

- **优先级**: P2
- **状态**: ✅ 已实现 (v1.3)
- **描述**: EPUB 内部链接（脚注、交叉引用）可点击跳转
- **验收标准**:
  - 识别同章节锚点跳转
  - 识别跨章节链接跳转
  - 外部 URL 不打开（防止意外跳出）

### FR-010: 保留 EPUB 原始样式

- **优先级**: P2
- **状态**: ✅ 已实现 (v1.3)
- **描述**: 对于样式精美的 EPUB，可保留其原始 CSS
- **验收标准**:
  - 设置开关控制
  - 开启时注入最小覆盖样式（仅控制溢出和分页）
  - 原始字体、颜色、布局保留

## 3. 非功能需求

### NFR-001: 性能

- 章节加载 < 1秒（普通章节）
- 翻页响应 < 100ms
- 内存占用 < 150MB

### NFR-002: 兼容性

- 最低 Android 7.0 (API 24)
- 必须在 HarmonyOS 4 WebView 上正确分页
- 不依赖 Google Play Services

### NFR-003: 离线可用

- 所有功能完全离线
- 无网络请求
- EPUB 文件存储在应用内部

### NFR-004: APK 大小

- 目标 < 10MB（当前 ~5.3MB）
