# Copilot Instructions — 林中路阅读器

## 项目简介

Android EPUB 阅读器，Kotlin，WebView 渲染，scrollTop 分页引擎。
详细文档见 `docs/` 目录。

## 开发必读

1. **每次代码修改后必须递增版本号** — `app/build.gradle.kts` 中 `versionCode` +1, `versionName` 更新
2. **构建命令**: `.\build-release.ps1`
3. **核心文件**: `app/src/main/java/com/lisb/reader/ui/ReaderActivity.kt`（含 READER_JS）
4. **分页架构**: scrollTop + wrapper.clientHeight + 底部遮罩（不要用 CSS transform）
5. **HTML 骨架**: `<body><div#lisb-wrapper><div#lisb-content>内容</div></div></body>`

## 技术约束

- JDK 17, AGP 8.7.3, Kotlin 1.9.24, Gradle 8.10.2
- compileSdk=36, minSdk=24
- 必须兼容 HarmonyOS 4 WebView
- 修改 CSS/HTML 时必须同步更新三种模式（普通/保留样式/TTS）
