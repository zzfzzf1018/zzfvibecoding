# Changelog

All notable changes to this project will be documented in this file.
The format is based on [Keep a Changelog](https://keepachangelog.com/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [0.1.0] - 2026-05-31

### Added
- Cross-platform (Windows + Android) Flutter scaffolding.
- GitHub OAuth Device Flow login.
- Create-private-repo + clone bootstrap of a notebook.
- Filesystem note repository (markdown + plain text).
- Markdown editor with live preview and inline image rendering.
- Manual sync + configurable auto-sync (Settings → Sync).
- Smart paste: plain text and HTML → markdown (image paste via injectable
  clipboard provider).
- Settings: theme, font family, font size, paragraph spacing, font color.
- Unit tests for entities, repository, markdown converter and sync service.
- PowerShell + Bash dual-target build scripts.
- Design spec, user manual and developer guide.
